FAQ

Comment changer les droits sur Linux?

La commande chmod permet de modifier les droits et permissions.

Que veut dire la propriété NaN en Js?

La propriété NaN signifie que la variable en question n’est pas un nombre. (Not a Number)

Est-il possible de modifier un commit?

La commande git commit --amend permet de modifier facilement le commit le plus récent. Elle vous permet de combiner les changements stagés avec l'ancien commit au lieu de créer un commit totalement nouveau.

Comment mettre à jour Ubuntu?

Dans un terminal, tapez "sudo apt update" (Ubuntu va chercher les nouveaux paquets), ensuite tapez "Sudo apt upgrade" pour lancer la maj.

Dois-je mettre la balise de fin php ?> ?

Lorsque PHP traite un fichier, il cherche les balises d'ouverture et de fermeture ( ) qui délimitent le code qu'il doit interpréter. De cette manière, cela permet à PHP d'être intégré dans toutes sortes de documents, car tout ce qui se trouve en dehors des balises ouvrantes / fermantes de PHP est ignoré.
